Cоздаем модель машинного обучения для предсказания покупательской активности
на примере интернет-магазина "В один клик".

 Описание проекта.
Отчет интернет-магазина «В один клик» за прошлый период показал, что активность покупателей начала снижаться.

Цель заказчика — удерживать активность постоянных клиентов с помощью персонализированных предложений.

Задачи:

построить модель, которая предсказывает вероятность снижения покупательской активности клиента в следующие три месяца.
выделить сегменты покупателей и разработать для них персонализированные предложения.


Используемые кодировщики: OrdinalEncoder для порядковых категорий, OneHotEncoder для остальных категориальных признаков.

Используемые скейлеры: StandardScaler, MinMaxScaler, RobustScaler.

Из четырех моделей с разными гиперпараметрами лучше всех справилась модель, основанная на методе опорных векторов, на втором месте - метод K ближайших соседей, на третьем месте - древесная модель,

ROC-AUC лучшей модели на тренировочных данных: 0.91 ROC-AUC лучшей модели на тестовых данных: 0.92
